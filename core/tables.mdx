## Default properties

Each table has different default properties that will appear as columns. All of the tables have the same design—the only difference is
the type of data you are looking at.

## Manipulating tables

For any table, you can shrink or grow columns, move columns around, or use the <em>action bar</em> to manipulate the table:
add properties, hide properties, or sort.

## Sort

Using the action bar, you can use the dropdown menu to sort properties within the table.

## Add or remove properties

Using the action bar, you can use the dropdown menu to add or remove properties within the table.

## Drag columns around or change column width 


- Grab the column’s header with your mouse to move it around
- Hover you mouse on the property header, you’ll notice the icon appear where you can expand or contract the width of the column; if you double click, we will auto-size it.
